## API Endpoint Map

```
REST API ENDPOINTS

POST /extract
  Upload image â†’ Extract text â†’ Return JSON
  Request: multipart/form-data (file)
  Response: { source_filename, raw, patient, insurance, clinician, consent_text }

GET /api/forms
  Get all records
  Response: [{ id, original_filename, created_at, updated_at }, ...]

POST /api/forms
  Create new record
  Request: { original_filename, extracted_json }
  Response: { id, original_filename, extracted_json, created_at, updated_at }

GET /api/forms/:id
  Get single record
  Response: { id, original_filename, extracted_json, created_at, updated_at }

PUT /api/forms/:id
  Update record
  Request: { extracted_json }
  Response: { id, original_filename, extracted_json, created_at, updated_at }

DELETE /api/forms/:id
  Delete record
  Response: { message: "Form deleted successfully" }

  Get statistics
  Response: { total_forms: 42 }

GET /health
  Health check
  Response: { status: "ok" }
```
